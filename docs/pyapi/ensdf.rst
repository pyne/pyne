***************************************
ENSDF File Support -- :mod:`pyne.ensdf`
***************************************
Pyne should nominally contain support for reading and writing 
Evaluated Nuclear Structure Data Files (ENSDF).  However, currently
only parsing out half-life data is supported.

.. currentmodule:: pyne.ensdf

All functionality may be found in the ``ensdf`` module::

    from pyne import ensdf
