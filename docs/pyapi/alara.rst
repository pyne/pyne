====================================================
ALARA Input and Output Interfaces -- :mod:`pyne.alara`
====================================================

.. currentmodule:: pyne.alara

There is a module for reading and writing ALARA input to and from PyNE Mesh
objects. The functionality of the module can be obtained by importing as such::

    from pyne import alara

You may also want to consult the userguide page :ref:`usersguide_alara`.
